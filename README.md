# 🎬 Movie App

A modern movie discovery web application built with **React.js**, **Vite**, **React Router**, and **TMDB API**. Browse popular movies, search for your favorite titles, and manage a personalized favorites list.

## ✨ Features

🔍 Search movies by title

🎥 Browse popular movies

❤️ Add and remove movies from favorites

🗂️ Favorites page with persistent state

⚡ Fast and responsive user interface

📱 Mobile-friendly design

🌐 Real-time movie data from TMDB API

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🛣️ React Router DOM
* 🎯 Context API
* 🎨 CSS3
* 🎬 TMDB API

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/movie-explorer-react.git
```

Navigate to the project directory:

```bash
cd movie-explorer-react
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_TMDB_API_KEY=your_tmdb_api_key
```

## 🏗️ Build for Production

```bash
npm run build
```
